# pip 套件管理軟體

## 安裝 pip

*python 2*

```shell
sudo apt-get install python-pip python-dev build-essential
sudo pip install --upgrade pip
sudo pip install --upgrade virtualenv
```

*python 3*

```shell
sudo apt-get install python3-pip
```


## ipython

terminal 互動式 python 套件

```shell
$ pip install ipython
```

